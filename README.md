# 🎫 Online Ticket Booking Platform

A modern full-stack ticket booking platform where users can browse available bus and train tickets, view ticket details, book seats, manage bookings, and access a personalized dashboard.

## 🌐 Live Demo

🔗 Live Site: https://online-ticket-booking-platform-coral.vercel.app

🔗 GitHub Repository: https://github.com/Arshad-Hossain/online-ticket-booking-platform

---

## 📖 Project Overview

Online Ticket Booking Platform is a responsive web application designed to simplify ticket purchasing and management.

Users can:

- Browse available tickets
- View detailed ticket information
- Book tickets securely
- Manage personal bookings
- Track booking history
- Access dashboard analytics

Vendors can:

- Add new tickets
- Manage ticket inventory
- Monitor bookings
- Track sales and revenue

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Arshad-Hossain/online-ticket-booking-platform.git
```

### Navigate to Project

```bash
cd online-ticket-booking-platform
```

### Install Dependencies

```bash
npm install
```

### Setup Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_SERVER_URL=your_server_url

MONGODB_URI=your_mongodb_uri

BETTER_AUTH_SECRET=your_auth_secret

BETTER_AUTH_URL=http://localhost:3000
```

### Run Development Server

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
